# ADPCM/リズム音源出力シミュレートビットクラッシャーVST『2686VFX』

## 目的

- 2686VのFXパネル部分をビットクラッシャープラグインとして独立、VST3エフェクトとして作成する
  - リズム音源・ADPCMで使用するビット深度・サンプリングレートで出力
  - 深度ビットとサンプリングレートは各音源YM2608B相当とするが、他の深度ビットとサンプリングレート(最大32bit/96kHz)なども可能にする

## 規格

- VST3(VST SDK最新バージョン)

## OS

- Windows(x64)

## 開発環境

- Microsoft Visual Studio 2026 Community
- Microsoft Visual Studio Code
  - CMakeを使用した環境構築を必要とした場合に使用

## 基本仕様

- 2686Vと同じソリューションで管理
- VST3(64bit)とスタンドアロンを供給
- 基本機能は、2686VのFXパネルと同じ
  - 入力がDAWトラック内のINSERT/SENDエフェクト欄への出力音声に変わる
  - スタンドアロン版では、PCの音声入力を変換する
- ユーザープリセットとしてファイルに保存できる
- プリセットブラウザパネルを作成する

  - 2686Vと共通
